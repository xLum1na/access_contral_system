# 智能门禁系统硬件选型规格书

## 1. 主控单元 (MCU)

| 项目           | 规格参数                                     | 备注                                        |
| :------------- | :------------------------------------------- | :------------------------------------------ |
| **型号**       | ESP32-P4                                     | 乐鑫新一代高性能RISC-V MCU                  |
| **处理器**     | 双核 RISC-V HP CPU @ 400MHz + LP CPU @ 40MHz | HP核运行AI/UI，LP核负责低功耗待机           |
| **内存**       | 内部SRAM 768KB + 支持外挂 PSRAM (最大32MB)   | 32MB PSRAM                                  |
| **显示接口**   | MIPI DSI (1-lane, 最高1Gbps/lane)            | 原生硬件支持，无需桥接芯片                  |
| **摄像头接口** | MIPI CSI (2-lane) + 硬件ISP                  | 支持800W像素输入，内置JPEG/H.264编码器      |
| **AI加速**     | 向量指令集 + 外部NPU扩展接口                 | 可运行轻量级人脸识别模型（如MobileFaceNet） |
| **连接性**     | Wi-Fi 6 (802.11ax) + Bluetooth 5.4 (LE/Mesh) | 支持Matter协议，便于智能家居/楼宇集成       |
| **安全特性**   | Secure Boot, Flash Encryption, HMAC, AES/TLS | 满足门禁数据加密与防篡改需求                |
| **封装**       | QFN68 / BGA                                  | 注意PCB布局需考虑高速信号完整性             |

> ⚠️ **关键提示：** ESP32-P4 目前处于量产初期，开发板可选 **ESP32-P4-Function-EV-Board**。务必确认所选模组已引出 MIPI-DSI 和 MIPI-CSI 引脚，部分精简版模组可能未引出。

## 2. 显示与触控单元

| 项目         | 规格参数                         | 适配说明                                     |
| :----------- | :------------------------------- | :------------------------------------------- |
| **屏幕型号** | Raspberry Pi 7" Touch Display V1 | 工业级成熟模组，供应链稳定                   |
| **分辨率**   | 800 × 480 IPS                    | P4的MIPI-DSI原生支持此分辨率                 |
| **显示接口** | 15-pin 1.0mm FPC MIPI DSI        | 需定制FPC转接板适配P4开发板/底板             |
| **触控芯片** | FT5406 (I²C地址: 0x38)           | ESP-IDF 已有 `esp_lcd_touch_ft5x06` 驱动组件 |
| **桥接芯片** | TC358762 (屏幕内置)              | P4直连MIPI DSI，**无需额外桥接**             |
| **背光控制** | ATTINY88 (屏幕内置)              | 通过I²C或GPIO控制亮度，P4可直接对接          |
| **可视角度** | 85°/85°/85°/85° (全视角IPS)      | 适合壁挂式门禁安装                           |
| **工作温度** | -20°C ~ 70°C                     | 满足室内/半室外门禁环境要求                  |

> 🔧 **适配要点：**
>
> - RPi 7寸屏的FPC引脚定义与标准MIPI-DSI不完全一致，**必须设计转接板**，将电源(3.3V/1.8V)、背光EN、I²C(TP+ATTINY)正确映射到P4底板。
> - ATTINY88 主要用于背光管理和屏幕初始化，P4可通过I²C与其通信调节亮度；若仅需固定亮度，可硬件上拉EN引脚简化设计。
> - ESP-IDF v5.3+ 已包含 `esp_lcd_mipi_dsi` 和 `esp_lcd_touch_ft5x06` 官方驱动，可直接使用LVGL构建UI。

## 3. 摄像头单元

| 项目           | 规格参数                              | 适配说明                                               |
| :------------- | :------------------------------------ | :----------------------------------------------------- |
| **传感器**     | IMX219  (Jetson Nano兼容系列)         | 800万像素，MIPI CSI 2-lane                             |
| **分辨率**     | 3280×2464 @ 15fps / 1920×1080 @ 30fps | 门禁场景建议使用1080p@30fps模式                        |
| **视场角**     | 120° (对角线)                         | 覆盖0.5m~3m通行距离，适合壁挂安装                      |
| **接口**       | 15-pin 1.0mm FPC MIPI CSI             | 与RPi/Jetson Nano摄像头排线兼容                        |
| **红外滤光片** | IRCUT 双滤光片切换                    | **门禁必备！** 白天彩色+夜间红外黑白，保证暗光人脸采集 |
| **补光灯**     | 850nm IR LED × 2~4颗                  | 配合IRCUT实现夜视，避免红光暴露                        |
| **ISP处理**    | P4内置硬件ISP                         | 支持自动曝光/白平衡/降噪/宽动态(WDR)                   |
| **活体检测**   | 单目+IR双目活体                       | 利用可见光+红外双通道实现基础活体防伪                  |

> ⚠️ **关键风险提示：**
>
> - **CSI引脚兼容性：** Jetson Nano摄像头的FPC引脚定义与ESP32-P4的MIPI-CSI **不一定完全兼容**。务必核对Data+/Data-/CLK/GND/I²C/PWDN/RESET引脚映射，可能需要定制转接FPC。
> - **P4 ISP配置：** ESP-IDF提供 `esp_cam_sensor` 组件，但IMX219/IMX415的寄存器配置需手动移植。建议参考乐鑫官方 `esp32-p4-camera-example` 进行适配。

## 4. 外围电路与接口规划

| 模块         | 推荐方案                                      | 用途                                 |
| :----------- | :-------------------------------------------- | :----------------------------------- |
| **音频**     | ES8311 Codec (I2S + I²C)                     | 语音提示、开门铃声、告警音          |
| **门锁控制** | 继电器(MOSFET隔离) + 门磁检测GPIO             | 干/湿节点输出，支持电磁锁/电插锁     |
| **存储**     | MicroSD卡槽 (SDMMC)                           | 本地日志缓存、离线人脸库备份         |
| **电源**     | 12V DC输入 → TPS5430/Buck → 5V/3.3V/0-12V可调 | P4核心电压1.2V，MIPI/摄像头需独立LDO |
| **调试**     | USB-C (UART/JTAG)                             | 固件烧录、日志输出、JTAG调试         |

## 5. 引脚定义表

表格



| 功能模块       | ESP32-P4 GPIO | 外设引脚/信号        | 电平/协议   | 备注                              |
| :------------- | :------------ | :------------------- | :---------- | :-------------------------------- |
| **MIPI DSI**   | 板上芯片固定  | DSI_D0±/D1±/CLK±     | MIPI 1-lane | 连接RPi 7寸屏转接板               |
| **触控I²C**    | GPIO7GPIO8    | FT5406 SDA/SCL       | I²C 3.3V    | 地址0x38，需外接4.7kΩ上拉         |
| **背光I²C**    | GPIO7/GPIO8   | ATTINY88 SDA/SCL     | I²C 3.3V    | 与触控共用I²C总线，地址不同       |
| **MIPI CSI**   | 板上芯片固定  | CSI_D0±/D1±/CLK±     | MIPI 2-lane | 连接IMX219转接FPC，注意差分对等长 |
| **CAM_I²C**    | GPIO7/GPIO8   | IMX219 SDA/SCL       | I²C 1.8V    | 传感器寄存器配置                  |
| **继电器驱动** | GPIO1         | AO3400 Gate          | 3.3V GPIO   | 串1kΩ限流 + 10kΩ下拉防上电误触发  |
| **音频 I2S**   | GPIO12 (BCK)  | ES8311 BCLK          | I2S 3.3V    | 位时钟                             |
|                | GPIO13 (MCK)  | ES8311 MCLK          | I2S 3.3V    | 主时钟                             |
|                | GPIO10 (WS)   | ES8311 LRCK          | I2S 3.3V    | 左右声道时钟                       |
|                | GPIO9 (DOUT)  | ES8311 DIN           | I2S 3.3V    | 数据输出 (播放)                    |
|                | GPIO48 (DIN)  | ES8311 DOUT          | I2S 3.3V    | 数据输入 (录音)                    |
|                | GPIO11 (PA)   | ES8311 PA_EN         | 3.3V GPIO   | 功放使能                           |
| **音频 I²C**   | GPIO7/GPIO8   | ES8311 SDA/SCL       | I²C 3.3V    | 地址0x18，共用I²C总线              |
| MicroSD        | GPIO39-44     | D0/D1/D2/D3/CMD/CLK  | SDMMC 3.3V  | 4-bit SDMMC 模式                   |

> 📌 **引脚分配注意事项：**
>
> 1. MIPI DSI/CSI 为高速差分信号，GPIO编号仅为示意，**实际必须以ESP32-P4数据手册中指定的MIPI专用引脚为准**，不可随意替换。
> 2. I²C总线（触控+背光）建议预留跳线焊盘，便于调试时分离排查。
> 3. 继电器驱动GPIO务必选择支持PWM输出的引脚，便于后续实现软启动/缓吸合功能延长触点寿命。
> 4. 所有对外接口（门磁、按钮、读卡器）均需加TVS/ESD保护器件，防止现场静电损坏P4。

## 6. 开发资源参考

- **ESP-IDF版本：** ≥ v5.3（P4支持从v5.3开始）
- **官方示例：** `examples/peripherals/mipi_dsi`, `examples/peripherals/camera`, `examples/gui/lvgl`
- **人脸算法：** ESP-WHO v3.x（支持P4，含MobileFaceNet检测+识别+活体）
- **UI框架：** LVGL v8.x/v9.x + SquareLine Studio可视化设计
- **原理图参考：** ESP32-P4-Function-EV-Board 开源硬件设计
